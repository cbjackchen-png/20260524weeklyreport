上週關鍵事件回顧

週一 (5/18): 油價在112美元震盪，費半跌2.47%，市場屏息等待Nvidia財報。川普拒絕伊朗最新提案並稱停火協議「命懸一線」，布蘭特油價盤中最高112美元後，川普宣布因波斯灣盟友請求暫緩對伊朗軍事行動，油價部分回落。S&P 500微跌0.07%，道瓊上漲159點，費半跌2.47%（財報前觀望）。ServiceNow大漲8.78%（BofA重啟覆蓋給予買進）、Dominion Energy漲9.44%（NextEra宣布全股票收購）。10年期殖利率盤中一度升至4.63%創波段新高。

週二 (5/19): 美債殖利率升破4.66%創波段新高，美股連三日收跌，市場升息定價急劇升溫。S&P 500跌0.67%，道瓊跌322點，30年期殖利率一度升至2007年以來高位，FedWatch顯示年底前升息概率達81%。Home Depot Q1財報小幅超標。原物料（-2.27%）、通訊服務（-1.58%）、非必要消費（-1.32%）領跌；醫療保健（+1.09%）、能源（+1.03%）防禦性板塊抗跌。Marvell +4.35%（Evercore重申優於大盤）。

週三 (5/20): 油價暴跌5.6%、殖利率急速回落，Nvidia財報大超預期，美股四日來首度強勁反彈。S&P 500漲1.08%，道瓊漲645點重返50,000點，那指漲1.54%，費半漲4.49%。Nvidia Q1營收816億美元（年增85%，超預期790億），Q2指引910億美元。AMD漲8.1%、Intel漲7.4%。美伊談判傳入「最終階段」，布蘭特暴跌5.6%至105美元。FOMC會議紀要顯示多數官員傾向刪除寬鬆偏向措辭。

週四 (5/21): Nvidia股價利多出盡跌1.77%，Walmart財報超標但指引保守跌7.3%，量子運算族群爆發。S&P 500微漲0.17%，道瓊漲276點再創歷史新高，費半漲1.28%，Nvidia跌1.77%（「例行性超標」新鮮感遞減）。Walmart Q1亮眼但Q2 EPS指引低於預期跌7.3%。川普政府透過CHIPS法案撥款20億美元支持量子運算九家公司，D-Wave Quantum漲33%、Rigetti漲30%。Spotify漲13%（投資人日強勁前景）。

週五 (5/22): 沃什宣誓就任Fed主席，沃勒鷹派講話確定升息定價，道瓊創歷史新高、標普八連紅。S&P 500漲0.37%，道瓊漲294點再創歷史新高50,579點，費半漲1.99%，高通飆升11.6%（Stellantis多年智慧汽車合作）。聯想財報強勁（Q4年增27%）帶動戴爾漲16%、惠普漲15%，兩者均創歷史新高。沃勒發表鷹派演講：「不能再排除未來加息的可能性」，貨幣市場完全定價2026年加息25個基點。密西根消費者信心44.8創歷史新低，通膨預期4.8%。

本週市場脈動 Overview

聯準會鷹派轉向與新主席上任，升息預期驟升：新任聯準會主席凱文·沃什正式於白宮宣誓就職，而偏鴿派的理事沃勒發表強烈鷹派言論，主張刪除政策聲明中的寬鬆傾向措辭，並明確表示「不能再排除未來升息的可能性」。市場預期劇烈翻轉，交易員徹底打消今年降息的念頭，並預期2026年底前升息25個基點的機率達到完全定價。美國5月密西根大學消費者信心指數跌至44.8的歷史新低，消費者對未來一年的通膨預期攀升至4.8%，顯示通膨壓力已深度植入社會預期。

AI狂熱疊加超級IPO潮，標普連漲八週創紀錄：Nvidia Q1財報再度大幅超標（營收816億美元，年增85%，Q2指引910億美元），在高殖利率環境下為AI科技多頭提供最關鍵的基本面背書，帶動標普500連續第八週收紅，為2023年以來最長週線連漲紀錄。資本市場IPO熱潮全面引爆，SpaceX正式向SEC提交IPO申請，最快6月12日掛牌，目標估值達1.75兆美元；美銀分析師警告，若這些巨型IPO落地，美股集中度將輕易突破48%的歷史泡沫極值，超越1920年代、1990年代科技泡沫的所有歷史紀錄。

美伊停火協議迎重大進展，國際油價週跌逾8%：中東局勢迎來關鍵轉機，川普宣布美伊已基本談成和平協議，且霍爾木茲海峽將重新開放。儘管伊朗官方隨後澄清川普發言不完整，強調海峽管理權仍歸伊朗所有，且雙方在核問題與制裁解除的細節上仍有嚴重分歧，但地緣政治風險降溫的預期已促使油價週跌逾8%，為本輪衝突以來最大週跌幅。能源通膨壓力的階段性緩解，是本週美股得以八連紅的重要支撐因素之一。

展望下週（5月25日起，週一美股因陣亡將士紀念日休市）：

關鍵通膨指標PCE與GDP數據公佈：下週四（28日）美國將公佈聯準會最青睞的通膨指標——4月核心PCE物價指數，以及第一季實際GDP修正值。在近期CPI與PPI雙雙意外火熱的背景下，若PCE增速超出預期，恐進一步強化年內升息的押注，並推升美債殖利率。

科技巨頭與財報壓軸登場：美股Q1財報季進入尾聲，焦點將轉向企業軟體端——Snowflake、邁威爾科技（Marvell）、Salesforce及戴爾科技將陸續揭曉業績，邁威爾作為AI ASIC與光互連技術龍頭，其數據中心業務成長將是檢驗AI基礎設施需求的重要指標。

聯準會官員密集談話：隨著升息預期升溫，下週包含紐約聯儲主席威廉斯、理事鮑曼等多位聯準會高層將發表談話，市場將密切尋找內部是否已由鷹派佔據上風的政策線索。

分析師洞察：

  • 因果鏈：本週美股的核心驅動邏輯由「通膨風險 × AI財報驗證 × 地緣緩解」三股力量的動態博弈構成。週一至週二，油價高企（布蘭特最高112美元）→ 通膨預期強化 → 美債殖利率週二升至4.66%波段高位 → 市場對高估值成長股進行估值修正，費半連跌兩日。週三迎來逆轉：美伊談判傳進展 → 布蘭特單日暴跌5.6%至105美元 → 通膨預期降溫 → 殖利率回落至4.57% → 股市全面強勁反彈，道瓊漲645點重返50,000點。週四至週五，Nvidia財報超標（816億美元，年增85%）提供AI需求的最強基本面背書 → 費半年初至今+72.28%，標普累計八連紅。然而，Nvidia盤後股價僅小幅反應，顯示「例行性超標」的市場驚喜效應正在遞減，信號意義已轉為「確認已知預期」而非「創造新預期」。
  • 風險思辨：本週最值得深思的市場訊號，不是指數的八連紅，而是兩個「財報超標卻股價下跌」的異常現象：Nvidia Q1超標且Q2指引910億美元，但週五收跌1.9%、全週跌4.43%；Walmart Q1亮眼但Q2指引保守，跌7.3%。這兩個訊號共同揭示了當前市場的結構性矛盾——AI硬體需求的強度已被充分定價，而消費端則開始傳遞高油價侵蝕購買力的早期警訊。美股集中度48%（含SpaceX等超大型IPO落地後）已超越歷史上所有重大泡沫峰值，這一極端集中本身就是最大的系統性風險。
  • 前瞻性洞察：下週四的核心PCE數據是決定短期市場方向的最關鍵指標。若PCE年增速超過4%（接近CPI的3.8%水準），市場對沃勒鷹派立場的定價將從「警告」升級為「行動」——這將引發殖利率再度攀升並對高估值板塊構成新一輪壓力。另一個重要觀察框架是油價：若美伊談判本週達成正式協議，布蘭特可能快速回落至90美元以下，這將是通膨降溫、殖利率回落、市場再度追逐成長股的最強催化劑；反之若談判再度破裂，油價重返110美元以上將使整個「八連紅」的基礎動搖。

分析師洞察：

  • 因果鏈：本週板塊輪動格局呈現出「全面性反彈但由少數板塊主導」的特徵，XSD半導體週漲9.89%（年初至今+87.79%，再創歷史高位）是最強的單週板塊表現，驅動力明確：Nvidia財報超標 → AMD漲8.1%、Intel漲7.4%、高通漲11.6% → 費半週漲5.30%、年初至今+72.28%。同時值得注意的是本週罕見的「防禦 + 進攻同步上漲」格局——XLU公用事業週漲3.37%、XLV醫療週漲3.30%，與XSD同列前三，這反映了市場內部的分裂：部分資金繼續追逐AI動能，另一部分資金在BofA牛熊指標觸及8.0賣出訊號後，悄悄向防禦性板塊轉移。能源（XLE）本週僅漲0.08%，是本週油價急跌（週跌逾8%）壓制的直接體現，然而年初至今仍高達+33.06%，說明地緣溢價即使部分回吐，結構性油價上移的趨勢並未逆轉。
  • 風險思辨：XLC通訊服務（-0.53%，年初至今-1.92%）是本週唯一收跌的板塊，且是全年表現最差的板塊之一。核心原因是廣告科技與媒體股的動能分歧——Google週跌3.48%、Meta週跌0.65%，市場開始懷疑高AI投資是否能轉化為等比例的廣告收入成長。這一疑慮與「Nvidia財報超標但股價下跌」的邏輯一脈相承：市場已從「相信故事」進入「驗證變現」的階段，所有AI主題標的都需要用實際的每股盈餘成長來支撐當前的估值溢價。
  • 前瞻性洞察：從板塊結構看，目前最值得追蹤的輪動方向是「半導體 → 軟體/應用層」的傳導。本週戴爾漲16%、惠普漲15%（聯想財報帶動PC硬體需求確認）、Workday漲5%，顯示AI需求的廣度正在從晶片擴散至整個IT基礎設施鏈。若下週PCE數據不超預期、油價穩定在100美元以下，這一輪動有機會延續；若殖利率再度急升，最脆弱的是估值最高的XSD，而年初至今+87.79%的位置，任何負面衝擊都可能引發劇烈的去槓桿。

分析師洞察：

  • 因果鏈：本週債市走勢是本輪最具戲劇性的週度波動——10年期殖利率週初升至4.66%（波段新高），週末收在4.56%（週跌3.7個基點），30年期週初觸及5.13%後收在5.06%（週跌6.4個基點）。這一「先升後回落」的格局背後有兩個清晰的驅動機制：①升段（週一至週二）：油價高企（布蘭特達112美元）→ 通膨預期強化 → 市場對Fed年底前升息的押注升至81% → 殖利率快速攀升；②降段（週三起）：美伊談判傳入「最終階段」→ 布蘭特單日暴跌5.6% → 通膨風險溢價急速回吐 → 中長端殖利率回落，帶動股市全面反彈。值得特別關注的是，週五沃勒鷹派演講（「不能再排除未來升息」）與沃什就任幾乎同步發生，2年期美債殖利率單日上行4個基點至4.12%，為2025年2月以來最高，貨幣市場已完全定價2026年加息25個基點。
  • 風險思辨：30年期殖利率維持在5.06%，雖較前週5.13%略有回落，但仍處於2007年以來的高位水準，且月漲16.6個基點。這一水準意味著什麼？它是「股票定價」的隱性錨點——當30年期美債提供5%的無風險收益，股票溢價的合理性需要企業盈利成長率高於5%+風險溢價（通常為3-4%）的基準，即每年至少需要8-9%的盈利成長才能支撐當前估值。費城半導體年初至今+72.28%的漲幅，其背後的隱含成長預期已遠超這一合理水準。歷史規律顯示，新任Fed主席就任後3個月殖利率平均上行55個基點——若此規律重演，10年期年底可能升至5%以上，那將是對高估值成長股的結構性衝擊。
  • 前瞻性洞察：下週四的核心PCE（聯準會最青睞的通膨指標）與GDP修正值的組合，將是債市的最關鍵分水嶺。若PCE年率突破4%（從上月3.8%繼續上行），市場將被迫重新評估「升息幅度」而非「升息是否發生」——這意味著從「加息一次25bps」升級為「加息1-2次」的定價，將驅動30年期殖利率再度測試5.1%的前高。反之，若PCE保持穩定在3.5-3.8%區間，短暫的緩和窗口可能帶動成長股的反彈，但結構性高利率環境不會因一次數據改變。

3. 美股IPO集中度達48%——超大型IPO潮的市場集中風險

分析觀點因果鏈：Cloud積壓訂單4,600億（近乎翻倍）→ Gemini 3生態整合 → AI變現路徑從雲端擴散至整體生態系。主要風險是廣告收入對AI投資的支撐能力——Google週跌3.48%反映市場對廣告與AI ROI之間缺口的憂慮；反壟斷訴訟是長期估值壓制因素。若下週PCE超預期，高估值成長股估值壓力可能延續。

Astera Labs (ALAB) - 展望樂觀

PCIe Gen6滲透率快速提升，Scorpio X成下一成長引擎Astera Labs作為純AI連接晶片公司，產品線全面放量。PCIe Gen6滲透率快速提升，新一代Scorpio X交換器預期於2026年超越前代成為主力成長引擎，解決AI推論與Agentic workload的記憶體與GPU互聯瓶頸。自訂晶片與UALink業界標準預計2027年導入，加上光通訊（NPO/CPO）的矽光子與光纖耦合技術佈局，將打開下一個數十億美元級別的市場機會。確認與Nvidia合作開發NVLink Fusion，並在Azure等Hyperscaler獲得CXL記憶體連接大單。
分析觀點因果鏈：AI推論需求爆發 → PCIe Gen6互聯瓶頸 → Scorpio X交換器成主力 → 2027年UALink標準導入 → 數十億美元市場機會。主要風險是估值已相當充分定價，且客戶集中度高（依賴少數Hyperscaler大客戶）；若任一大客戶調整ASIC投資時程，季度財報可能出現明顯落差。

Keysight Technologies (KEYS) - 展望樂觀

AI帶動有線測試爆發，週期性復甦已明確展現Keysight Technologies儘管無線測試需求復甦較平緩，但受惠於AI加速帶動的網路與資料中心測試需求爆發，強勁的有線測試需求有效抵銷整體波動。2Q26已展現明確的週期性復甦，BofA上調目標價並看好其未來有能力超越原本設定的5-7%長期營收成長模型。資料中心基礎設施的升級測試需求，為Keysight提供了跨越傳統無線週期的增量成長動力。
分析觀點因果鏈：AI資料中心建設加速 → 有線測試需求爆發 → 抵銷無線測試週期疲弱 → 週期性復甦確認。主要風險是若AI資本支出週期出現中斷或延後，有線測試需求的支撐可能比預期更脆弱；而無線市場（5G建設放緩）的拖累效應可能在有線需求消化後重新浮現。下週KEYS本身無重大財報，但需密切追蹤Marvell與戴爾等相關供應鏈業者的數據中心業務指引。
